Trabalhando com Listas

Listas Ordenadas

Através da tag <ol>, pode-se criar uma lista ordenada ("ordered list"). Por exemplo:

  1. Item número 1
  2. Item número 2
  3. Item número 3
  4. Item número 4

Cada item da lista deve estar entre tags <li> ("list item"). Quando os <li> estão dentro de uma tag <ol>, são list items ordenados de acordo com a ordem de aparição no HTML.

As listas ordenadas <ol> contém o parâmetro type="", que determina a forma como a lista é ordenada, sendo uma lista numérica, alfabética ou em números romanos. Por exemplo:

  1. Item número 1
  2. Item número 2
  3. Item número 3
  4. Item número 4

Acima, uma lista ordenada alfabética. Abaixo, uma lista em algarismos romanos:

  1. Item número 1
  2. Item número 2
  3. Item número 3
  4. Item número 4

Importante ressaltar que o parâmetro type="", embora ainda funcional, deve ser considerado obsoleto por referir-se ao estilo de ordenação da lista. Como é possível fazer a mesma mudança através de CSS, é recomendado usar CSS para alterar o tipo de ordenação da lista.

Listas Não-Ordenadas

As listas não-ordenadas utilizam a tag <ul> ("unordered list").

  • Item número 1
  • Item número 2
  • Item número 3
  • Item número 4

Da mesma forma, listas não-ordenadas também podem ter seus estilos alterados através do mesmo parâmetro type="", porém, é recomendado que se use CSS para isto.

  • Item número 1
  • Item número 2
  • Item número 3
  • Item número 4
  • Item número 1
  • Item número 2
  • Item número 3
  • Item número 4

Juntando listas

  1. Tópico número 1

Listas de Definição

As listas mais únicas são as listas de definição, usando a tag <dl> ("definition list"). Cada item de uma lista de definição usa duas tags; uma tag para o ítem (<dt>, "definition term") e uma tag para sua descrição (<dd>, "definition description").
