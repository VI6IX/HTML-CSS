Principais Formatações

Negrito

Nessa frase temos um termo em negrito usando a tag <b>. A tag <b> não é semântica, portanto, o ideal é utilizar a tag <strong> para aplicar negrito.

Itálico / Ênfase

Nesta frase, há um termo em itálico através da tag <i>, que não é semântica. A tag semântica para itálico e/ou ênfase é <em>, que contém o significado de ênfase para o HTML.

Texto marcado

Pode-se marcar um pedaço do texto através da tag <mark>, cujo efeito é similar ao de um marca-texto. Esta é uma tag semântica com significado de marcação

Texto grande e pequeno

Neste parágrafo, há um texto pequeno. Embora haja uma tag semântica — <small> — para reduzir o tamanho de um texto, o qual seria, por exemplo, uma bula ou copyright, não há tag semântica para aumentar o tamanho do texto, e isto deve ser feito através do CSS.

Texto deletado

Podemos marcar um texto como excluído, indicando que embora deva ser lido, não deva ser considerado, como uma tese que já foi descartada ou uma mudança de opiniões.

Texto inserido

Pode-se também marcar um texto como inserido, enfatisando que este foi adicionado posteriormente de quando o texto foi originalmente escrito.

Texto sobrescrito

Usando a tag <sup>, pode-se marcar um tetxo como sobrescrito, tornando o texto menor e movido para cima, como no caso de uma potência em matemática; x6.

Texto subscrito

Similar ao texto sobrescrito, com a tag <sub>, podemos marcar um texto como subescrito, tornando-o menor e posicionado abaixo, como na quantidade de átomos em química; H20
