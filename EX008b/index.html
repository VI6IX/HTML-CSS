Código-fonte

O comando documento.getElementById('teste') é escrito em linguagem JavaScript. Note como o método ficou marcado com fonte monoespaçado, o que é feito através da tag <code>. Bem útil para menções de funções, variáveis, etc.

Para blocos de código, entretanto, é necessário colocar a tag <code> dentro de outra tag; <pre>. Por exemplo:

        
    var iVar = 0
    for (iVar = 0; iVar <= 100; iVar += 1) {
        print(iVar);
        print(`Valor de 1 incrementado à ${iVar}`);
    }
    if (iVar == 100) {
        print('Contagem realizada.')
    }

Citações

Um texto pode ser marcado como citação através da tag <q>, que introduz aspas.

Citações completas

Para blocos de citações, usa-se a tag <blockquote>, que desloca e isola o texto.

Abreviações

Podemos marcar siglas e/ou abreviações usando a tag <abbr>, que é uma tag semântica, logo, deve ser usada exclusivamente no HTML e não no CSS.
